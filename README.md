# Language Words  
  
This project helps keep and test words from learning language.  
The tech stack contains Angular 4, Node.js, Express, MongoDB.  
  
### Build  
  
git clone https://ledamint@bitbucket.org/ledamint/language-words.git  
cd language-words/  
npm i --production  
npm run build  
http://localhost:3000
